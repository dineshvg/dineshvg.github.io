# Dinesh Gangatharan Portfolio

A modern, high-performance portfolio website built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

## 🚀 Deployment to GitHub Pages

This project is configured to deploy automatically using **GitHub Actions**.

### 1. Push your code to GitHub
If you haven't already, push your local changes to your GitHub repository:
```bash
git add .
git commit -m "Update portfolio content and deployment guide"
git push origin main
```

### 2. Enable GitHub Actions for Deployment
1. Go to your repository on GitHub.com.
2. Click on **Settings** (top tab).
3. Click on **Pages** (left sidebar).
4. Under **Build and deployment** > **Source**, select **GitHub Actions** from the dropdown menu.

### 3. Verify Deployment
- Once selected, GitHub will automatically trigger the deployment workflow defined in `.github/workflows/deploy.yml`.
- You can monitor the progress under the **Actions** tab.
- Your site will be live at `https://dineshvg.github.io/`.
